# Forum_project
Forum flask website project

## Installation Linux
```bash
# install 
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```

## Run Linux
```bash
# run 
export FLASK_ENV=development
flask run
```

## Installation Windows
```bash
# install  
python -m virtualenv .
.\scripts\activate
pip install -r requirements.txt
```
## Run Windows
```bash
# run 
set FLASK_ENV=development
set FLASK_APP=main.py
flask run
```

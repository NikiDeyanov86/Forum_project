# Forum_project
Forum flask website project

## Instalation
```bash
# install Linux
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```
# install Windows 
python -m virtualenv .
.\scripts\activate
pip install -r requirements.txt

## Run
```bash
# run Linux
export FLASK_ENV=development
flask run
```
# run Windows 
set FLASK_ENV=development
set FLASK_APP=main.py
flask run
